# MODL Examples

This directory contains comprehensive examples demonstrating each of the 7 core constructs of MODL.

## Files

1. **01_entity.modl** - Entity construction examples
2. **02_relation.modl** - Relation construction examples
3. **03_constraint.modl** - Constraint construction examples
4. **04_transform.modl** - Transform construction examples
5. **05_pattern.modl** - Pattern construction examples
6. **06_composition.modl** - Composition construction examples
7. **07_query.modl** - Query construction examples

Each file demonstrates the syntax, semantics, and practical usage of its respective construct.
